| Command              | Description                             |
| -------------------- | --------------------------------------- |
| `tap`                | Lists valid commands                    |
| `tap <command>`      | Lists valid subcommands for `<command>` |
| `tap <command> -h`   | Writes help information for `<command>` |
|                      |                                         |
| `tap run <testplan>` | Run `<testplan>`                        |
|                      |                                         |
